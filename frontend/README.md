# AI Code Editor - Frontend

This is the React-based frontend for the AI Code Editor.

## Key Components
- `App.jsx`: Main interface featuring a split-pane layout with Monaco Editor and Chat Sidebar.
- `index.css`: Custom design system featuring glassmorphic panels and a dark theme.
- Integration: Communicates with the Node.js backend on port 5000.

## Commands
- `npm install`: Install dependencies.
- `npm run dev`: Start development server.

Refer to the [Root README](../README.md) for full project details and backend setup.
